# digital-methodologies

La raíz del deploy (`index.html`) muestra el MVP HTML funcional de la app antiestafas, en una versión mobile-first centrada solo en el teléfono y con una presentación visual refinada.

El proyecto Android real del MVP antiestafas vive en `android-app/`.

El prototipo HTML interactivo del MVP vive en `mvp-android-prototype.html`.
